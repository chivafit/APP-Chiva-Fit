🟣 Yampi (Checkout)

Pedidos e carrinhos abandonados são enviados automaticamente via webhook da Yampi para o Supabase.
Configure em Yampi → Configurações → Webhooks usando a URL da função yampi-webhook no Supabase.

Não é necessário informar token aqui no CRM — a integração é feita 100% pelo backend.
